# Coffee Sales Dashboard

## Objective
Analyze coffee sales data to identify sales trends, product performance, store-wise revenue, and customer purchasing patterns.

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Dashboard Design

## Dataset Columns
- transaction_id
- transaction_date
- transaction_time
- transaction_qty
- store_id
- store_location
- product_id
- unit_price
- product_category
- product_type
- product_detail

## Key Analysis Performed
- Daily & Monthly Sales Analysis
- Store-wise Revenue Analysis
- Product Category Performance
- Top Selling Products
- Sales Quantity Tracking
- Peak Sales Time Analysis

## Skills Used
- Data Cleaning
- Pivot Tables
- Excel Charts
- Dashboard Creation
- Data Visualization
- KPI Analysis

## Outcome
Developed an interactive Excel dashboard to visualize coffee sales performance, identify business trends, and support data-driven decision-making.
